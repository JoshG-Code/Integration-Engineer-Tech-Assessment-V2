Project Vector

A physics-driven 2D platformer built in Unity

About the Game

Project Vector is a fast-paced 2D platformer centered around gravity manipulation mechanics and precision movement gameplay. Players navigate through dynamic environments using directional gravity shifts, momentum-based traversal, and physics-driven puzzles.

Developed in Unity, this project was created as a collaborative game development initiative where I served as both Project Lead and Lead Programmer.
